# PHelper
### 这是一个基于Node.js的爬虫项目，主要用来爬取[pixiv-插画网站](http://pixiv.net)上的高分图片
主要代码分布在getURL和download文件夹中
##### getURL：按照ID逐个遍历pixiv上的图片，按一定规则筛选后提取页面中的图片URL，写入文件中
##### download：读取url文件下载图片
